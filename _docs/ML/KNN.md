# K-최근접 이웃 알고리즘 (KNN, K-Nearest Neighbor Algorithm)
- KNN 알고리즘은 input_data로부터 거리가 가까운 'k'개의 다른 데이터의 레이블을 참조하여 분류하는 알고리즘
- 거리를 측정할 때 **유클리디안 거리** 계산법을 활용
- KNN 간단하지만 이미지 처리, 글자 인식, 영화나 음악 추천 알고리즘 등 많은 분야에서 사용됨
